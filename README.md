# pixiv-utils
Pixiv工具类

# 使用方法
```xml
<project>
    <repositories>
        <repository>
            <id>org.gin</id>
            <name>common-utils</name>
            <url>https://agmonk.github.io/pixiv-utils/maven-repo/</url>
        </repository>
    </repositories>
    
    <dependency>
        <groupId>org.gin</groupId>
        <artifactId>pixiv-utils</artifactId>
        <version>1.0.0</version>
    </dependency>
</project>
```
